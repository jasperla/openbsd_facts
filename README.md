openbsd_facts
=============

This is a potpouri of various facts specific to OpenBSD.

Available facts
---------------

- diskcount
  `hw.diskcount`

- disknames
  `hw.disknames`

- lidsuspend
  `machdep.lidsuspend`

- fusefs_open_devices
  `vfs.fuse.fusefs_open_devices`

- led_blind
  `machdep.led_blink`

- byteorder
  `hw.byteorder`

- osrevision
  `kern.osrevision`

- securelevel
  `kern.securelevel`

- livelocks
  `kern.netlivelocks`

- buffercache
  `kern.bufcachepercent`

- sensors
  `hw.sensors`
